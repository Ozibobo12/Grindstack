## ⚡ Build in 3 Steps

### Option A — Android Studio (easiest)
1. Open **Android Studio** → *Open* → select this folder
2. Wait for Gradle sync to complete
3. Go to **Build → Generate Signed Bundle / APK → APK**
4. Choose a keystore (or create one), click **Finish**
5. Your APK is in `app/build/outputs/apk/release/`

### Option B — Command Line
```bash
# macOS / Linux
chmod +x gradlew
./gradlew assembleDebug

# Windows
gradlew.bat assembleDebug
```
Output: `app/build/outputs/apk/debug/app-debug.apk`

### Option C — GitHub Actions (auto-build)
1. Push this project to a GitHub repo
2. The included `.github/workflows/build.yml` will build automatically
3. Download the APK from the **Actions → Artifacts** tab
